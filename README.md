# Audiovisual Information Processing

This repository contains laboratory works for the course **Audiovisual Information Processing**.

The main code for all laboratory works is located in the [`labs.ipynb`](labs.ipynb) notebook.  
Separate report notebooks and additional datasets are also included in the repository.

## Description

The project includes practical work related to audiovisual information processing and computer vision tasks.  
The laboratory works cover data analysis, image and signal processing, feature extraction, visualization, and practical experiments using Python notebooks.
